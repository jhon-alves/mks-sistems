## Features

A few of the things you can do with MKS Sistems:

* View products in homepage
* Add products in cart
* Remove products in cart

## Build Process

- Clone or download the repo
- `yarn` to install dependencies
- `yarn dev` to execute project

## Technologies

- TypeScript
- React
- Redux Toolkit
- Styled-components
- Material UI
